# data/laws/

Akoma Ntoso XML fajlovi izabranih zakona.

**Vlasnik:** Član 1
**Celina:** 1

## Naming konvencija

Naziv fajla = `id` zakona. Aplikacija ga koristi u URL-ovima (`/laws/{id}`).

Primer: `kz.xml` za Krivični zakonik.

## Šta ide ovde

Minimum: KZ čl. 260–268 + relevantne definicije iz opšteg dela (čl. 112).
Ako se referenciraju drugi zakoni iz odluka (Zakon o zaštiti životne sredine i sl.), i njih dodati.

Sve mora biti potpuno anotirano (videti spec, Celina 1).
